SpoilerAL 6.1 修正パッチ用 example.dll の関数


void HelloWorldA(LPCSTR s)

  概要
    MessageBoxA のタイトルを "HelloWorldA" とし、引数をメッセージボックスに表示します。
  引数
    s ... メッセージボックスに表示するテキスト


void HelloWorldW(LPCWSTR s)

  概要
    MessageBoxW のタイトルを "HelloWorldW" とし、引数をメッセージボックスに表示します。
  引数
    s ... メッセージボックスに表示するテキスト


void HelloWorldVA(LPCSTR format, ...)

  概要
    MessageBoxA のタイトルを "HelloWorldVA" とし、引数を書式化しメッセージボックスに表示します。
    format に続く引数は、可変引数となり vsnprintf で書式化されます。
  引数
    format ... メッセージボックスに表示するテキスト


void HelloWorldVW(LPCWSTR format, ...)

  概要
    MessageBoxA のタイトルを "HelloWorldVW" とし、引数を書式化しメッセージボックスに表示します。
    format に続く引数は、可変引数となり _vsnwprintf で書式化されます。
  引数
    format ... メッセージボックスに表示するテキスト


DWORD Add32(DWORD a, DWORD b)

  概要
    32ビットの２つの引数を加算します。
  引数
    a ... 加数
    b ... 加数
  戻り値
    加算された値が返ります。


DWORD Sub32(DWORD a, DWORD b)

  概要
    被減数から減数を減算します。
    引数および戻り値はいずれも32ビットです。
  引数
    a ... 被減数
    b ... 減数
  戻り値
    減算された値が返ります。
